Purpose 
the api is a discretionary, variable 
component of compensation. Combined 
with the base salary, it provides 
competitive total cash compensation 
when both business and individual 
performance targets are achieved.

Structure 
swiss Re operates a target api (tapi)  
system along with a performance 
management framework that provides 

a tapi is set based on multiple factors, 
but primarily on the role being performed 
and market benchmarks. similar to  
the determination of the base salary,  
the employee’s total compensation and 
overall pay-mix are taken into account, 
when setting the tapi. the possible 
payout for the api ranges from 0 to  
2x tapi.

Settlement 
api is generally settled in cash. When the 
total api level for an employee exceeds  
a pre-defined amount, the award is split 
into two components: an immediate cash 
incentive payment (cash api) and a 
deferred api (Vai).
